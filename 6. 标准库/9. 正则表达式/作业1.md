1. 写一个正则表达式，匹配手机号
```js
^1[0-9]{10}$
^1\d{10}$

```
11位，第一位是1

2. 姓名必须是3-6位的中文
```js
^[\u4e00-\u9FA5]{3,6}$
```
3. 密码必须是6-12位的字符，只能包含数字、字母、下划线
```js
^[0-9a-Z]{6,12}$
^\w{6,12}$
```
4. 写一个正则表达式，匹配邮箱

xxxxxx@xxxxx.xxxx.xxxx
```js
^[0-9A-z]{5,20}@[0-9A-z]{2,10}(\.[0-9A-z]{2,5})+$
```
5. 匹配一个座机号

xxx-xxxxxxxx

前面：1-3个数字
后面：4-8个数字
^[0-9]{1,3}-[0-9]{4,8}$
6. 匹配一个正数
^[0-9]+(\.[0-9]+)?$
1. 匹配一个小数
^-?[0-9]+.[0-9]+$
2. 匹配一个整数
^-?[0-9]+$
